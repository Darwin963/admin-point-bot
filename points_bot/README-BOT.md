# AI Fixer Web (مصحح الأخطاء)

مجلد `web` يحتوي على تطبيق ويب بسيط (واجهة + خادم Express) يسمح لك بإرسال وصف لمشكلة أو كود ويطلب اقتراح إصلاح من OpenAI.

خطوات التشغيل:

1. انتقل للمجلد `web`:

```bash
cd "c:/Users/DELL/OneDrive/Desktop/Admin Point Bot/web"
```

2. ثبت الحزم:

```bash
npm install
```

3. أنشئ ملف `.env` في نفس المجلد وضع مفتاح OpenAI (إن وجد):

```
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini   # اختياري
```

ملاحظة: لا أستطيع تزويد خدمة "مجانية" للوصول إلى نماذج OpenAI. إن رغبت باستخدام بوت مجاني محلياً، يمكننا توجيهك لكيفية تشغيل نموذج مفتوح المصدر (مثل llama.cpp / ggml) وإيصال الخادم به.

4. شغّل الخادم:

```bash
npm start
```

ثم افتح المتصفح على http://localhost:3000

في الواجهة، يمكنك اختيار ملف من المشروع (bot.py, etc.) وتحميله لإصلاح الأخطاء فيه عند تعطل المشروع.

ماذا بعد؟
- إن أردت أتكفل بربط نموذج مفتوح المصدر محلي أو مساعدتك بضبط مفتاح OpenAI لديك.
